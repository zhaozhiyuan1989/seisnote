mseed2ascii¶

软件来源

https://github.com/iris-edu/mseed2ascii

适用平台

Linux、macOS

推荐阅读

暂无

更新日期

2021-01-27

将 mseed 格式的波形数据转为 ascii 格式，不需要元信息文件。

安装¶

目前源码在 GitHub 上公开。

$ wget https://github.com/iris-edu/mseed2ascii/archive/v2.6.tar.gz
$ tar -xvf v2.6.tar.gz
$ cd mseed2ascii-2.6
$ make && mv mseed2ascii ~/bin
